# cep-answer

Aplicação de linha de comando para buscar endereços a partir de um CEP brasileiro.

## Tecnologias

- Node.js
- Axios
- API ViaCEP

## Instalação

1. Clone o repositório:
   git clone https://github.com/seu-usuario/cep-answer.git

2. Instale as dependências:
   npm install

## Como usar

Execute o comando abaixo e digite o CEP quando solicitado:

   node main.js

Exemplo de saída:

   Endereço encontrado:
   Rua:        Praça da Sé
   Bairro:     Sé
   Cidade:     São Paulo
   Estado:     SP
